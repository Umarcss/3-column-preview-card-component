# Frontend Mentor – 3-column preview card component

This is a solution to the [3-column preview card component challenge](https://www.frontendmentor.io/challenges/3column-preview-card-component-pbmryBmKD) on Frontend Mentor.

## Overview

A responsive component featuring three columns (Sedans, SUVs, Luxury) with SVG icons, brief descriptions, and styled call-to-action buttons. Designed with accessibility and mobile responsiveness in mind, leveraging custom fonts and modern CSS/SASS.

![Component Screenshot](./images/Screenshot%20From%202025-11-03%2004-13-47.png)

## Built With
- HTML5
- SASS (SCSS)
- CSS3 (compiled from SCSS)
- Flexbox & Responsive Design
- [Google Fonts](https://fonts.google.com/): Big Shoulders Display, Lexend Deca

## Getting Started

1. **Clone this repo:**
```bash
git clone https://github.com/your-username/3-column-preview-card-component.git
```
2. **Open `index.html` in your browser** to view the component.

3. **(Optional) Rebuild CSS from SCSS:**
   - If you want to modify styles, edit files in `sass/` and compile with:
   ```bash
   sass sass/main.scss css/style.css
   ```
   - Requires [Dart Sass](https://sass-lang.com/install) (`sass`)

## Folder Structure
